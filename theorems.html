Important Theorems

Key Theorems with Proofs for AKTU DSTL Exams

Unit I: Sets & Relations

De Morgan's Laws for Sets Unit I

Theorem: For any sets A and B:

(i) (A ∪ B)' = A' ∩ B'
(ii) (A ∩ B)' = A' ∪ B'

Proof of (i): (A ∪ B)' = A' ∩ B'

1
Let x ∈ (A ∪ B)'
2
Then x ∉ (A ∪ B)    [Definition of complement]
3
So x ∉ A and x ∉ B    [Definition of union]
4
Therefore x ∈ A' and x ∈ B'    [Definition of complement]
5
Hence x ∈ A' ∩ B'    [Definition of intersection]
∎
The reverse inclusion follows similarly. Therefore (A ∪ B)' = A' ∩ B'

Equivalence Classes Partition Theorem IMPORTANT

Theorem: Let R be an equivalence relation on set A. Then the equivalence classes of R partition A.

That is: (1) A = ⋃[a] for all a ∈ A, and (2) [a] ∩ [b] = ∅ or [a] = [b]

Proof:

1
Union covers A: For any a ∈ A, a ∈ [a] (by reflexivity), so A ⊆ ⋃[a]. Clearly ⋃[a] ⊆ A.
2
Disjoint or Equal: Suppose [a] ∩ [b] ≠ ∅. Let c ∈ [a] ∩ [b].
3
Then aRc and bRc. By symmetry, cRb. By transitivity, aRb.
4
For any x ∈ [a]: aRx. Since aRb, by transitivity bRx, so x ∈ [b]. Hence [a] ⊆ [b].
∎
Similarly [b] ⊆ [a]. Therefore [a] = [b]. ∎
Corollary:
Number of equivalence classes = Number of partitions

Unit II: Propositional Logic

Contrapositive Equivalence Unit II

Theorem: p → q ≡ ¬q → ¬p

Proof using Truth Table:

p q p → q ¬q → ¬p
TTTT
TFFF
FTTT
FFTT
∎
Columns 3 and 4 are identical, proving the equivalence.

De Morgan's Laws for Logic Unit II

Theorem:
(i) ¬(p ∧ q) ≡ ¬p ∨ ¬q
(ii) ¬(p ∨ q) ≡ ¬p ∧ ¬q

Proof of (i): ¬(p ∧ q) ≡ ¬p ∨ ¬q

1
¬(p ∧ q) is TRUE when p ∧ q is FALSE
2
p ∧ q is FALSE when at least one of p or q is FALSE
3
This means ¬p is TRUE or ¬q is TRUE
∎
Which is exactly ¬p ∨ ¬q. Verify with truth table for complete proof.

Unit III: Boolean Algebra & Lattices

Lattice Absorption Laws IMPORTANT

Theorem: In any lattice (L, ∧, ∨), for all a, b ∈ L:

(i) a ∨ (a ∧ b) = a
(ii) a ∧ (a ∨ b) = a

Proof of (i):

1
We know a ∧ b ≤ a (definition of meet)
2
Also a ≤ a (reflexivity)
3
So a is an upper bound of {a, a ∧ b}
4
Since a ∨ (a ∧ b) is LUB of {a, a ∧ b}, we have a ∨ (a ∧ b) ≤ a
∎
But a ≤ a ∨ (a ∧ b) always. Therefore a ∨ (a ∧ b) = a. ∎

Complement Uniqueness in Distributive Lattice IMPORTANT

Theorem: In a bounded distributive lattice, if an element has a complement, it is unique.

Proof:

1
Suppose a has two complements b and c in lattice L.
2
Then a ∧ b = 0, a ∨ b = 1, a ∧ c = 0, a ∨ c = 1
3
b = b ∧ 1 = b ∧ (a ∨ c) = (b ∧ a) ∨ (b ∧ c) = 0 ∨ (b ∧ c) = b ∧ c
4
Similarly: c = c ∧ (a ∨ b) = (c ∧ a) ∨ (c ∧ b) = b ∧ c
∎
Therefore b = b ∧ c = c. Complement is unique! ∎

Modular Lattice Characterization Unit III

Theorem: A lattice L is modular if and only if it does not contain the pentagon N₅ as a sublattice.
Pentagon N₅:
The pentagon N₅ = {0, a, b, c, 1} where 0 < a < 1, 0 < b < c < 1, and a is incomparable with b and c.
This is the smallest non-modular lattice.

Distributive Lattice Characterization Unit III

Theorem: A lattice L is distributive if and only if it contains neither the pentagon N₅ nor the diamond M₃ as a sublattice.
Corollary:
Every distributive lattice is modular (but not vice versa).

Every Finite Lattice has Max and Min Elements IMPORTANT

Theorem: Every finite lattice has a greatest element (1) and a least element (0).

Proof:

1
Let L = {a₁, a₂, ..., aₙ} be a finite lattice.
2
Since L is a lattice, every pair has a join (LUB).
3
Consider: 1 = a₁ ∨ a₂ ∨ ... ∨ aₙ
4
This exists by repeated application of binary join, and is the greatest element.
∎
Similarly, 0 = a₁ ∧ a₂ ∧ ... ∧ aₙ is the least element. ∎

Duality Principle Unit III

Theorem: In Boolean algebra, if an identity holds, its dual (obtained by swapping + and ·, and 0 and 1) also holds.
Example:
Original: A + 0 = A   ⟹   Dual: A · 1 = A
Original: A + 1 = 1   ⟹   Dual: A · 0 = 0

Unit IV: Group Theory

Lagrange's Theorem VERY IMPORTANT

Theorem: If H is a subgroup of finite group G, then |H| divides |G|.

Moreover, [G : H] = |G| / |H| (index of H in G)

Proof:

1
The left cosets of H in G partition G (proved separately).
2
All left cosets have the same cardinality as H (|aH| = |H| for all a ∈ G).
3
Let [G : H] = k be the number of distinct cosets.
4
Then |G| = k × |H| since cosets partition G.
∎
Therefore |H| divides |G|, and [G : H] = |G|/|H|. ∎
Corollaries:
1. Order of any element divides |G|
2. a|G| = e for all a ∈ G
3. Groups of prime order are cyclic
Example:
In ℤ₁₂, subgroup H = {0, 4, 8} has order 3, which divides 12 ✓
Index [ℤ₁₂ : H] = 12/3 = 4 cosets ✓

Group Homomorphism Properties IMPORTANT

Theorem: Let f: G → G' be a group homomorphism. Then:

(i) f(eG) = eG' (identity maps to identity)
(ii) f(a⁻¹) = [f(a)]⁻¹ (inverse maps to inverse)
(iii) f(aⁿ) = [f(a)]ⁿ for all n ∈ ℤ

Proof:

1
(i) f(e) = f(e·e) = f(e)·f(e)
2
Multiply both sides by [f(e)]⁻¹: e' = f(e) ✓
3
(ii) f(a)·f(a⁻¹) = f(a·a⁻¹) = f(e) = e'
4
So f(a⁻¹) is the inverse of f(a), i.e., f(a⁻¹) = [f(a)]⁻¹ ✓
∎
(iii) follows by induction using the homomorphism property. ∎

First Isomorphism Theorem Unit IV

Theorem: If f: G → G' is a group homomorphism, then:

G / ker(f) ≅ im(f)
Key Concepts:
• Kernel: ker(f) = {a ∈ G : f(a) = e'}
• Image: im(f) = {f(a) : a ∈ G}
• ker(f) is a normal subgroup of G

Euler's Theorem IMPORTANT

Theorem: If gcd(a, n) = 1, then aφ(n) ≡ 1 (mod n)

Proof:

1
Consider U(n) = {x ∈ ℤₙ : gcd(x, n) = 1} under multiplication mod n.
2
U(n) is a group of order φ(n).
3
Since gcd(a, n) = 1, we have a ∈ U(n).
4
By Lagrange's corollary: a|U(n)| = e in U(n).
∎
Therefore aφ(n) ≡ 1 (mod n). ∎
Fermat's Little Theorem (Special Case):
If p is prime and gcd(a, p) = 1, then ap-1 ≡ 1 (mod p)

Cyclic Group Generator Theorem Unit IV

Theorem: In a cyclic group ⟨a⟩ of order n, the element ak is a generator iff gcd(k, n) = 1.

Corollary: Number of generators = φ(n)

Proof:

1
Order of ak is n/gcd(k, n).
2
ak generates the group iff its order equals n.
3
n/gcd(k, n) = n iff gcd(k, n) = 1.
∎
Count of such k from 1 to n is φ(n). ∎
Example (ℤ₈):
Generators are 1, 3, 5, 7 (coprime to 8)
φ(8) = 4 generators ✓

Unit V: Graph Theory

Handshaking Lemma IMPORTANT

Theorem: In any graph G = (V, E), the sum of degrees equals twice the number of edges:

Σ deg(v) = 2|E|

Proof:

1
Each edge contributes exactly 2 to the total degree sum.
2
One for each endpoint of the edge.
∎
Therefore Σ deg(v) = 2|E|. ∎
Corollary:
The number of vertices with odd degree is always even.

Euler's Formula for Planar Graphs VERY IMPORTANT

Theorem: For any connected planar graph with V vertices, E edges, and F faces:

V - E + F = 2

Proof by Induction on Edges:

1
Base: Tree with n vertices has n-1 edges, 1 face. V-E+F = n-(n-1)+1 = 2 ✓
2
Induction: Add edge e connecting existing vertices.
3
E increases by 1, F increases by 1 (edge divides a face).
∎
V - (E+1) + (F+1) = V - E + F = 2 (preserved). ∎
Corollaries:
1. E ≤ 3V - 6 for simple planar graphs (V ≥ 3)
2. E ≤ 2V - 4 if no triangles
3. K₅ and K₃,₃ are not planar

Kuratowski's Theorem IMPORTANT

Theorem: A graph is planar if and only if it contains no subdivision of K₅ or K₃,₃.
Application:
• K₅: Complete graph on 5 vertices (non-planar)
• K₃,₃: Complete bipartite graph (non-planar)
• Any graph containing these (or their subdivisions) is non-planar

Eulerian Graph Characterization Unit V

Theorem: A connected graph has an Eulerian circuit if and only if every vertex has even degree.

A connected graph has an Eulerian path (but not circuit) if and only if it has exactly two vertices of odd degree.

Proof Sketch (Necessity):

1
An Eulerian circuit visits each edge exactly once.
2
At each vertex, we enter and leave the same number of times.
3
So each vertex must have even degree.
∎
Sufficiency proved by Fleury's algorithm. ∎

Pigeonhole Principle Unit V

Theorem: If n items are placed into k containers where n > k, then at least one container has more than one item.

Generalized: At least one container has ⌈n/k⌉ items.
Classic Example:
In a room of 13 people, at least 2 share a birth month (13 people, 12 months).
