5Groups: Define Abelian group.

Solution:

An Abelian group (or commutative group) is a group (G, *) that satisfies:

1. Closure: a * b ∈ G for all a, b ∈ G
2. Associativity: (a * b) * c = a * (b * c)
3. Identity: ∃e ∈ G such that a * e = e * a = a
4. Inverse: For each a ∈ G, ∃a⁻¹ such that a * a⁻¹ = e
5. Commutativity: a * b = b * a for all a, b ∈ G

Example: (ℤ, +) is an Abelian group.

Q4bProve De Morgan's Laws for Boolean Algebra: (i) (a + b)' = a' · b' (ii) (a · b)' = a' + b'

Solution:

De Morgan's Laws - Proof using Boolean Algebra Axioms

(i) Prove: (a + b)' = a' · b'

We need to show that (a + b)' is the complement of (a + b).
By definition of complement, x' is unique element such that:
• x + x' = 1
• x · x' = 0

Step 1: Show (a + b) + (a' · b') = 1
(a + b) + (a' · b')
= [(a + b) + a'] · [(a + b) + b'] ... Distributive
= [(a + a') + b] · [a + (b + b')] ... Associative + Commutative
= [1 + b] · [a + 1] ... Complement law
= 1 · 1 = 1 ✓ ... Identity law

Step 2: Show (a + b) · (a' · b') = 0
(a + b) · (a' · b')
= (a · a' · b') + (b · a' · b') ... Distributive
= (0 · b') + (a' · 0) ... Complement law
= 0 + 0 = 0 ✓ ... Identity law

∴ (a + b)' = a' · b' ∎


(ii) Prove: (a · b)' = a' + b'

Step 1: Show (a · b) + (a' + b') = 1
(a · b) + (a' + b')
= (a · b + a') + b' ... Associative
= [(a + a') · (b + a')] + b' ... Distributive
= [1 · (b + a')] + b' ... Complement
= (b + a') + b' ... Identity
= a' + (b + b') ... Commutative + Associative
= a' + 1 = 1 ✓ ... Complement + Identity

Step 2: Show (a · b) · (a' + b') = 0
(a · b) · (a' + b')
= (a · b · a') + (a · b · b') ... Distributive
= (a · a' · b) + (a · b · b') ... Commutative
= (0 · b) + (a · 0) ... Complement
= 0 + 0 = 0 ✓ ... Identity

∴ (a · b)' = a' + b' ∎

Alternative Proof using Duality:
Since we proved (a + b)' = a' · b', by principle of duality (interchanging + and ·), we get (a · b)' = a' + b'.

These laws are fundamental in digital circuit simplification!

Q5aUsing inference rules, prove: Premises: (i) ∀x [P(x) → Q(x)] (ii) ∀x [Q(x) → R(x)] (iii) ∃x P(x). Conclusion: ∃x R(x)

Solution:

Formal Proof using Predicate Logic Inference Rules

Given Premises:
1. ∀x [P(x) → Q(x)]
2. ∀x [Q(x) → R(x)]
3. ∃x P(x)

To Prove: ∃x R(x)

Proof:

Step 1: From premise 3 (∃x P(x)), by Existential Instantiation (EI):
Let c be a particular element such that P(c) is true.
4. P(c) ... EI on premise 3

Step 2: From premise 1 (∀x [P(x) → Q(x)]), by Universal Instantiation (UI) with x = c:
5. P(c) → Q(c) ... UI on premise 1

Step 3: From statements 4 and 5, by Modus Ponens:
6. Q(c) ... MP (4, 5)

Step 4: From premise 2 (∀x [Q(x) → R(x)]), by Universal Instantiation (UI) with x = c:
7. Q(c) → R(c) ... UI on premise 2

Step 5: From statements 6 and 7, by Modus Ponens:
8. R(c) ... MP (6, 7)

Step 6: From statement 8, by Existential Generalization (EG):
9. ∃x R(x) ... EG on statement 8 ✓

∴ ∃x R(x) is proved. ∎

Summary of Inference Rules Used:
• EI (Existential Instantiation): ∃x P(x) → P(c) for some c
• UI (Universal Instantiation): ∀x P(x) → P(c) for any c
• MP (Modus Ponens): p, p→q ⊢ q
• EG (Existential Generalization): P(c) → ∃x P(x)

This proof demonstrates the transitivity of implication in predicate logic!

Q6bLet G be a group and H be a subgroup of G. Prove that the left cosets of H in G partition G.

Solution:

Theorem: Left cosets of H partition G

Given:
• G is a group
• H is a subgroup of G
• Left coset: aH = {ah | h ∈ H} for a ∈ G

To Prove: The collection of all distinct left cosets {aH | a ∈ G} partitions G.

For partition, we must show:
1. Every element of G is in some coset (covering)
2. Two cosets are either identical or disjoint (no overlap)


Part 1: Every element is in some coset (Covering)

Let a ∈ G be any element.
Since H is a subgroup, e ∈ H (identity element).
Therefore, a = ae ∈ aH.

∴ Every element of G belongs to at least one left coset. ✓


Part 2: Two cosets are identical or disjoint

Let aH and bH be two left cosets.
We'll prove: If aH ∩ bH ≠ ∅, then aH = bH.

Assume: aH ∩ bH ≠ ∅
Then ∃ x ∈ aH ∩ bH

Since x ∈ aH: x = ah₁ for some h₁ ∈ H ... (1)
Since x ∈ bH: x = bh₂ for some h₂ ∈ H ... (2)

From (1) and (2): ah₁ = bh₂
⟹ a = bh₂h₁⁻¹

Now show aH ⊆ bH:
Let y ∈ aH, then y = ah for some h ∈ H
y = ah = (bh₂h₁⁻¹)h = b(h₂h₁⁻¹h)

Since H is a subgroup, h₂h₁⁻¹h ∈ H
∴ y ∈ bH
Thus aH ⊆ bH ✓

Similarly show bH ⊆ aH:
From ah₁ = bh₂, we get b = ah₁h₂⁻¹
Let z ∈ bH, then z = bh' for some h' ∈ H
z = bh' = (ah₁h₂⁻¹)h' = a(h₁h₂⁻¹h')
Since h₁h₂⁻¹h' ∈ H, z ∈ aH
Thus bH ⊆ aH ✓

∴ aH = bH

Conclusion:
If two cosets overlap, they must be identical.
Otherwise, they are disjoint.

∴ The left cosets of H partition G. ∎

Consequence (Lagrange's Theorem):
Since cosets partition G and all cosets have equal size |H|:
|G| = (number of cosets) × |H|
∴ |H| divides |G|

This is a fundamental result in group theory!

Q4aState and prove Lagrange's theorem for groups.

If G is a finite group and H is a subgroup of G, then |H| divides |G|.
Moreover: |G| = |H| × [G:H]
where [G:H] is the index (number of cosets).


Proof:

1. Left cosets of H partition G (disjoint and cover G)

2. All left cosets have same cardinality:
φ: H → aH defined by φ(h) = ah is a bijection
• Injective: ah₁ = ah₂ ⟹ h₁ = h₂
• Surjective: every ah ∈ aH is image of h
∴ |aH| = |H| for all a ∈ G

3. Let there be k distinct left cosets: a₁H, a₂H, ..., aₖH

4. Since G = a₁H ∪ a₂H ∪ ... ∪ aₖH (disjoint):
|G| = |a₁H| + |a₂H| + ... + |aₖH|
|G| = k × |H|

∴ |H| divides |G| ∎
