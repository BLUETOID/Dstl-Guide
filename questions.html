Set Theory Questions

1If A = {1, 2, 3, 4, 5} and B = {4, 5, 6, 7}, find A ∪ B, A ∩ B, A - B, and B - A.

Union combines all elements, intersection takes common, difference takes elements in first but not second.
A ∪ B = {1, 2, 3, 4, 5, 6, 7}
A ∩ B = {4, 5}
A - B = {1, 2, 3}
B - A = {6, 7}

2Find the power set of A = {a, b, c}. How many elements does P(A) have?

Power set contains all subsets, including empty set and the set itself. |P(A)| = 2^|A|
P(A) = {∅, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c}}
|P(A)| = 2³ = 8 elements

3Prove using set identities: (A ∩ B)' = A' ∪ B' (De Morgan's Law)

Show that an element is in LHS iff it's in RHS. Use definition of complement and intersection.
x ∈ (A ∩ B)' ⟺ x ∉ (A ∩ B)
⟺ ¬(x ∈ A ∧ x ∈ B)
⟺ (x ∉ A) ∨ (x ∉ B) [De Morgan in logic]
⟺ x ∈ A' ∨ x ∈ B'
⟺ x ∈ (A' ∪ B') ✓

4In a class of 60 students, 35 study Mathematics, 30 study Physics, and 10 study both. How many study neither?

Use Inclusion-Exclusion: |A ∪ B| = |A| + |B| - |A ∩ B|
|M ∪ P| = |M| + |P| - |M ∩ P|
|M ∪ P| = 35 + 30 - 10 = 55
Students studying neither = 60 - 55 = 5 students

5If |A| = 5 and |B| = 3, find the maximum and minimum values of |A ∪ B| and |A ∩ B|.

Consider when B ⊆ A (maximum overlap) and when A and B are disjoint (no overlap).
|A ∪ B|:
• Maximum = 5 + 3 = 8 (when disjoint)
• Minimum = 5 (when B ⊆ A)

|A ∩ B|:
• Maximum = 3 (when B ⊆ A)
• Minimum = 0 (when disjoint)

Reflexive: Yes ✓ - (1,1), (2,2), (3,3) all present
Symmetric: Yes ✓ - (1,2) and (2,1) both present
Transitive: Yes ✓ - Check all pairs, satisfied
R is an Equivalence Relation!

3Prove that an equivalence relation on a set A partitions A into disjoint equivalence classes.

Show: (1) Every element is in some class (reflexive), (2) Classes are either equal or disjoint.
1. Every a ∈ A is in [a] (reflexive: a ~ a)
2. If [a] ∩ [b] ≠ ∅, take x ∈ [a] ∩ [b]

5Determine if R = {(a,b) | a divides b} on ℤ⁺ is a partial order. Find LUB and GLB of {4, 6} in divisibility poset.

Partial order needs reflexive, antisymmetric, transitive. LUB = LCM, GLB = GCD for divisibility.
Partial Order Check:
• Reflexive: a | a ✓ (every number divides itself)
• Antisymmetric: If a | b and b | a, then a = b ✓
• Transitive: If a | b and b | c, then a | c ✓

Yes, divisibility is a partial order!

For {4, 6}:
• GLB (Greatest Lower Bound) = GCD(4,6) = 2
• LUB (Least Upper Bound) = LCM(4,6) = 12

6Is the poset ({1, 2, 3, 6, 12, 24}, |) a lattice? Justify your answer.

A lattice requires every pair of elements to have both a LUB (join) and GLB (meet) within the set.
Check for every pair:

In divisibility poset:
• GLB(a,b) = GCD(a,b)
• LUB(a,b) = LCM(a,b)

For all pairs in {1, 2, 3, 6, 12, 24}:
• GCD always exists and is in the set
• LCM always exists and is in the set (since they all divide 24)

Examples:
• LCM(2,3) = 6 ∈ set ✓
• LCM(3,12) = 12 ∈ set ✓
• GCD(12,24) = 12 ∈ set ✓

Yes, it is a lattice!

7Find the reflexive closure and symmetric closure of R = {(1,2), (2,3)} on A = {1,2,3}.

Reflexive closure: add all (a,a). Symmetric closure: add (b,a) for each (a,b).

Add (1,1), (2,2), (3,3)
r(R) = {(1,1), (2,2), (3,3), (1,2), (2,3)}

Symmetric Closure s(R):
Add (2,1) for (1,2), add (3,2) for (2,3)
s(R) = {(1,2), (2,1), (2,3), (3,2)}

3[AKTU 2021] Prove that for any two sets A and B: |A ∪ B| = |A| + |B| - |A ∩ B|

Partition A∪B into disjoint parts and count each element exactly once.
Proof (Inclusion-Exclusion for 2 sets):

Method 1: Partition Argument
A ∪ B can be partitioned into 3 disjoint regions:
1. A ∩ B (elements in both)
2. A - B (elements only in A)
3. B - A (elements only in B)

So: |A ∪ B| = |A - B| + |A ∩ B| + |B - A|

We know:
• |A| = |A - B| + |A ∩ B|
• |B| = |B - A| + |A ∩ B|

Adding: |A| + |B| = |A - B| + |B - A| + 2|A ∩ B|

Therefore: |A - B| + |B - A| = |A| + |B| - 2|A ∩ B|

Substituting:
|A ∪ B| = |A| + |B| - 2|A ∩ B| + |A ∩ B|
|A ∪ B| = |A| + |B| - |A ∩ B| ∎

1If |A| = 4 and |B| = 3, how many functions exist from A to B?

Each element in A can map to any element in B independently.
Number of functions = |B|^|A| = 3⁴ = 81 functions

2Let f(x) = 2x + 3 and g(x) = x². Find (g ∘ f)(x) and (f ∘ g)(x).

(g ∘ f)(x) = g(f(x)). Apply f first, then g.
(g ∘ f)(x) = g(f(x)) = g(2x+3) = (2x+3)² = 4x² + 12x + 9

(f ∘ g)(x) = f(g(x)) = f(x²) = 2x² + 3 = 2x² + 3

Note: g ∘ f ≠ f ∘ g (composition not commutative)

3Prove that f: ℝ → ℝ defined by f(x) = 3x - 7 is bijective and find f⁻¹(x).

For injective: f(a) = f(b) implies a = b. For surjective: solve y = 3x - 7 for x.
Injective: f(a) = f(b) ⟹ 3a - 7 = 3b - 7 ⟹ a = b ✓

Surjective: For any y ∈ ℝ, x = (y+7)/3 gives f(x) = y ✓

f⁻¹(x) = (x + 7)/3

4Prove that the composition of two bijective functions is bijective.

Show g ∘ f is injective and surjective separately, using that f and g are each bijective.
Let f: A→B and g: B→C be bijective.

Injective: (g∘f)(a) = (g∘f)(b)
⟹ g(f(a)) = g(f(b))
⟹ f(a) = f(b) [g injective]
⟹ a = b [f injective] ✓

Surjective: For c ∈ C,
∃b ∈ B: g(b) = c [g surjective]
∃a ∈ A: f(a) = b [f surjective]
So (g∘f)(a) = g(f(a)) = g(b) = c ✓

2[AKTU 2022] Prove the following Boolean identity: (A + B)(A + C) = A + BC

Use distribution, then simplify using A + AB = A.
Proof:

LHS: (A + B)(A + C)

Step 1: Apply distribution
= A·A + A·C + B·A + B·C

Step 2: Apply A·A = A (Idempotent)
= A + AC + AB + BC

Step 3: Factor out A
= A(1 + C + B) + BC

Step 4: Apply 1 + X = 1
= A·1 + BC

Step 5: Apply A·1 = A
= A + BC

LHS = RHS ∎

This is the "Absorption Law" extended form.

3[AKTU 2021] Define Big-O, Big-Ω, and Big-Θ notations. Prove that f(n) = 3n² + 2n + 1 = Θ(n²).

For Θ, prove both O(n²) AND Ω(n²) separately.
Definitions:

Big-O (Upper Bound):
f(n) = O(g(n)) if ∃c>0, n₀ such that f(n) ≤ c·g(n) ∀n ≥ n₀

Big-Ω (Lower Bound):
f(n) = Ω(g(n)) if ∃c>0, n₀ such that f(n) ≥ c·g(n) ∀n ≥ n₀

Big-Θ (Tight Bound):
f(n) = Θ(g(n)) if f(n) = O(g(n)) AND f(n) = Ω(g(n))


Proof: 3n² + 2n + 1 = Θ(n²)

Part 1: Show O(n²)
For n ≥ 1: 2n ≤ 2n², 1 ≤ n²
So: 3n² + 2n + 1 ≤ 3n² + 2n² + n² = 6n²
With c₁ = 6, n₀ = 1: f(n) = O(n²) ✓

Part 2: Show Ω(n²)
For n ≥ 1: 3n² + 2n + 1 ≥ 3n²
With c₂ = 3, n₀ = 1: f(n) = Ω(n²) ✓

Since f(n) = O(n²) and f(n) = Ω(n²):
f(n) = Θ(n²) ∎

4[AKTU 2020] If f: A → B and g: B → C are both bijections, prove that g ∘ f is also a bijection.

Bijection = injective + surjective. Prove both properties for composition.
Given: f: A → B and g: B → C are bijections.
To Prove: g ∘ f: A → C is a bijection.

Part 1: g ∘ f is injective (one-to-one)
Let (g ∘ f)(a₁) = (g ∘ f)(a₂)
⟹ g(f(a₁)) = g(f(a₂))
⟹ f(a₁) = f(a₂) (since g is injective)
⟹ a₁ = a₂ (since f is injective)
∴ g ∘ f is injective ✓

Part 2: g ∘ f is surjective (onto)
Let c ∈ C. We need a ∈ A such that (g ∘ f)(a) = c.
Since g is surjective, ∃b ∈ B with g(b) = c
Since f is surjective, ∃a ∈ A with f(a) = b
So (g ∘ f)(a) = g(f(a)) = g(b) = c ✓

Since g ∘ f is both injective and surjective:
g ∘ f is a bijection ∎

4State and explain Lagrange's Theorem.

1. Order of any element divides |G|
2. aᴵᴳᴵ = e for all a ∈ G
3. Groups of prime order have no proper subgroups
4. Groups of prime order are cyclic
