Set Theory

Definition: Set

A set is a well-defined collection of distinct objects called elements or members. "Well-defined" means we can determine if an object belongs to the set or not.

Set Notation Methods

MethodExampleDescription
Roster/TabularA = {1, 2, 3, 4, 5}List all elements
Set-BuilderA = {x | x ∈ ℤ, 1 ≤ x ≤ 5}Define by property
SemanticA = Set of first 5 natural numbersVerbal description

Important Number Sets

ℕ = Natural Numbers = {1, 2, 3, ...} or {0, 1, 2, ...}
ℤ = Integers = {..., -2, -1, 0, 1, 2, ...}
ℤ⁺ = Positive Integers = {1, 2, 3, ...}
ℚ = Rational Numbers = {p/q | p, q ∈ ℤ, q ≠ 0}
ℝ = Real Numbers (all points on number line)
ℂ = Complex Numbers = {a + bi | a, b ∈ ℝ}

Types of Sets

Empty Set (∅ or {})

Contains no elements.
Example: {x | x² = -1, x ∈ ℝ}

Singleton Set

Contains exactly one element.
Example: {7}, {∅}

Finite Set

Has countable, limited elements.
|A| = n (cardinality)

Infinite Set

Elements cannot be counted.
Example: ℕ, ℤ, ℝ

Universal Set (U)

Contains all elements under consideration in a context.

Power Set P(A)

Set of ALL subsets of A.
|P(A)| = 2^|A|

Example: Power Set

Find the power set of A = {1, 2, 3}

Solution: |A| = 3, so |P(A)| = 2³ = 8 subsets

P(A) = {∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}}

Subset Relations

Definitions

Subset (A ⊆ B): Every element of A is in B. (A can equal B)
Proper Subset (A ⊂ B): A ⊆ B and A ≠ B
Superset (A ⊇ B): B is a subset of A

Key Facts

• Empty set ∅ is subset of every set
• Every set is subset of itself
• A = B ⟺ (A ⊆ B and B ⊆ A)

Algebraic Laws of Sets

Identity Laws: A ∪ ∅ = A, A ∩ U = A
Domination: A ∪ U = U, A ∩ ∅ = ∅
Idempotent: A ∪ A = A, A ∩ A = A
Complement: A ∪ A' = U, A ∩ A' = ∅, (A')' = A
Commutative: A ∪ B = B ∪ A, A ∩ B = B ∩ A
Associative: (A ∪ B) ∪ C = A ∪ (B ∪ C)
Distributive: A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
De Morgan's: (A ∪ B)' = A' ∩ B', (A ∩ B)' = A' ∪ B'

Example: Prove De Morgan's Law

Prove (A ∪ B)' = A' ∩ B'

Proof:
Let x ∈ (A ∪ B)'
⟹ x ∉ (A ∪ B)
⟹ x ∉ A AND x ∉ B
⟹ x ∈ A' AND x ∈ B'
⟹ x ∈ A' ∩ B'
Similarly prove reverse. Hence (A ∪ B)' = A' ∩ B' ∎

Cartesian Product

Definition

A × B = {(a, b) | a ∈ A and b ∈ B}
|A × B| = |A| × |B|

Example

A = {1, 2}, B = {a, b, c}. Find A × B.

A × B = {(1,a), (1,b), (1,c), (2,a), (2,b), (2,c)}
|A × B| = 2 × 3 = 6

Relations

Definition: Relation

A relation R from set A to set B is a subset of A × B.
R ⊆ A × B. If (a, b) ∈ R, we write aRb.
A relation on set A means R ⊆ A × A.

Representing Relations

Matrix Representation (for R on {1,2,3} where R = {(1,1), (1,2), (2,3)})

        1   2   3
    ┌───────────────┐
  1 │  1   1   0   │
  2 │  0   0   1   │
  3 │  0   0   0   │
    └───────────────┘
    M[i][j] = 1 if (i,j) ∈ R

Properties of Relations

Reflexive

∀a ∈ A: (a, a) ∈ R
Matrix: All diagonal = 1
Example: ≤, =, divides

Irreflexive

∀a ∈ A: (a, a) ∉ R
Matrix: All diagonal = 0
Example: <, ≠, >

Symmetric

If (a,b) ∈ R ⟹ (b,a) ∈ R
Matrix: M = Mᵀ
Example: =, ≠, sibling

Antisymmetric

If (a,b) ∈ R and (b,a) ∈ R ⟹ a = b
Example: ≤, ≥, ⊆, divides

Asymmetric

If (a,b) ∈ R ⟹ (b,a) ∉ R
Irreflexive + Antisymmetric
Example: <, >

Transitive

If (a,b) ∈ R and (b,c) ∈ R ⟹ (a,c) ∈ R
Example: <, ≤, =, divides

Example: Identify Properties

R = {(1,1), (2,2), (3,3), (1,2), (2,1)} on A = {1, 2, 3}

Reflexive? Yes - (1,1), (2,2), (3,3) all present ✓
Symmetric? Yes - (1,2) and (2,1) both present ✓
Antisymmetric? No - (1,2) and (2,1) present but 1 ≠ 2 ✗
Transitive? Check: (1,2) and (2,1) ⟹ need (1,1) ✓ present. Yes ✓

Equivalence Relations

Definition

A relation R on set A is an equivalence relation if it is:
1. Reflexive - aRa for all a
2. Symmetric - aRb ⟹ bRa
3. Transitive - aRb and bRc ⟹ aRc

Equivalence Class

Definition

For element a, the equivalence class [a] = {x ∈ A | xRa}
All elements related to a form its equivalence class.

Partition Property

Equivalence classes partition the set A into disjoint subsets where:
• Every element belongs to exactly one class
• Union of all classes = A
• Different classes are disjoint

Example: Congruence Modulo 3

On ℤ, define aRb ⟺ a ≡ b (mod 3). Find equivalence classes.

Verify equivalence relation:
Reflexive: a - a = 0 = 3(0) ✓
Symmetric: 3|(a-b) ⟹ 3|(b-a) ✓
Transitive: 3|(a-b) and 3|(b-c) ⟹ 3|[(a-b)+(b-c)] = 3|(a-c) ✓

Equivalence Classes:
[0] = {..., -6, -3, 0, 3, 6, ...} (multiples of 3)
[1] = {..., -5, -2, 1, 4, 7, ...} (remainder 1)
[2] = {..., -4, -1, 2, 5, 8, ...} (remainder 2)

Partial Order & POSET

Definition: Partial Order

A relation R on set A is a partial order if it is:
1. Reflexive - aRa
2. Antisymmetric - aRb and bRa ⟹ a = b
3. Transitive - aRb and bRc ⟹ aRc

Definition: POSET

A POSET (Partially Ordered Set) is a pair (A, ≤) where ≤ is a partial order on A.

Key Terminology

TermSymbolDefinition
Comparablea ~ ba ≤ b OR b ≤ a
Incomparablea ∥ bNeither a ≤ b nor b ≤ a
Minimal-No element x with x < a
Maximal-No element x with a < x
Least/Minimum0a ≤ x for ALL x
Greatest/Maximum1x ≤ a for ALL x
Upper Bound of S-x ≥ s for all s ∈ S
Lower Bound of S-x ≤ s for all s ∈ S
LUB/Supremumsup(S)Least upper bound
GLB/Infimuminf(S)Greatest lower bound

Minimal vs Minimum

Minimal: No element below it (can have multiple)
Minimum: Below ALL elements (unique if exists)
Minimum is always minimal, but not vice versa!

Example: Divisibility POSET

Consider ({1,2,3,4,6,12}, |) where | means "divides"

Verify partial order:
Reflexive: a | a ✓
Antisymmetric: a | b and b | a ⟹ a = b ✓
Transitive: a | b and b | c ⟹ a | c ✓

Elements:
• Minimal = Minimum = 1
• Maximal = Maximum = 12
• LUB{2,3} = 6, GLB{4,6} = 2

Hasse Diagrams

Definition

A Hasse Diagram is a visual representation of a POSET where:
• Elements are vertices
• Greater elements are drawn higher
• Lines connect only covering pairs (no reflexive/transitive edges)

Covering Relation

Definition

a covers b (or b is covered by a) if b < a and there is NO c such that b < c < a.

Example Hasse Diagrams

Steps to Draw Hasse Diagram

  1. List all elements of the POSET
  2. Find the minimal elements (place at bottom)
  3. Find covering relations
  4. Draw edges for covering relations only
  5. Place greater elements higher (no arrows needed)

Example: Power Set

Draw Hasse diagram for (P({1,2,3}), ⊆)

P({1,2,3}) = {∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}, {1,2,3}}

Structure:
Level 0: ∅ (minimum)
Level 1: {1}, {2}, {3}
Level 2: {1,2}, {1,3}, {2,3}
Level 3: {1,2,3} (maximum)

Connect each set to sets with one more element.

Lattices

Definition: Lattice

A POSET (L, ≤) is a lattice if every pair of elements has:
• LUB (join): a ∨ b = sup{a, b}
• GLB (meet): a ∧ b = inf{a, b}

Types of Lattices

Bounded Lattice

Has both 0 (least) and 1 (greatest) elements.

Complemented Lattice

Bounded lattice where every element a has complement a' with a ∨ a' = 1, a ∧ a' = 0

Distributive Lattice

a ∧ (b ∨ c) = (a ∧ b) ∨ (a ∧ c)
a ∨ (b ∧ c) = (a ∨ b) ∧ (a ∨ c)

Boolean Lattice

Complemented + Distributive
Example: (P(S), ⊆)

Lattice Properties (Algebraic)

Idempotent: a ∨ a = a, a ∧ a = a
Commutative: a ∨ b = b ∨ a, a ∧ b = b ∧ a
Associative: (a ∨ b) ∨ c = a ∨ (b ∨ c)
Absorption: a ∨ (a ∧ b) = a, a ∧ (a ∨ b) = a

Example: Divisibility Lattice

Show ({1,2,3,6}, |) is a lattice and find 2 ∨ 3 and 2 ∧ 3.

For any a, b in this set:
• a ∨ b = LCM(a, b)
• a ∧ b = GCD(a, b)

2 ∨ 3 = LCM(2, 3) = 6
2 ∧ 3 = GCD(2, 3) = 1

Since every pair has both LCM and GCD in the set, it's a lattice ✓
