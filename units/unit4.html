Binary Operations

Definition

A binary operation * on set S is a function *: S × S → S
For a, b ∈ S, we write a * b instead of *(a, b).
Examples: +, ×, -, ÷ on numbers

Properties of Binary Operations

PropertyDefinitionExample
Closurea * b ∈ S for all a, b ∈ Sℤ under + is closed
Associativity(a * b) * c = a * (b * c)+ is associative
Commutativitya * b = b * a+ is commutative
Identity∃e: a * e = e * a = a0 for +, 1 for ×
Inverse∃a⁻¹: a * a⁻¹ = e-a for +, 1/a for ×

Groups

Definition: Group

A group (G, *) is a set G with binary operation * satisfying:
G1. Closure: a * b ∈ G for all a, b ∈ G
G2. Associativity: (a * b) * c = a * (b * c)
G3. Identity: ∃e ∈ G: a * e = e * a = a
G4. Inverse: ∀a ∈ G, ∃a⁻¹: a * a⁻¹ = a⁻¹ * a = e

Abelian Group

A group is Abelian (commutative) if a * b = b * a for all a, b ∈ G.

Not Groups - Examples

Why These Fail

• (ℤ, ×): No inverse for 2 (since 1/2 ∉ ℤ)
• (ℕ, +): No inverse for 1 (since -1 ∉ ℕ)
• (ℤ, -): Not associative: (5-3)-2 ≠ 5-(3-2)

Example: Verify Group

Show (ℤ₄, +₄) = ({0,1,2,3}, addition mod 4) is a group.

Closure: (a + b) mod 4 ∈ {0,1,2,3} ✓
Associativity: Inherited from integer addition ✓
Identity: 0 (a + 0 ≡ a mod 4) ✓
Inverses: 0⁻¹=0, 1⁻¹=3, 2⁻¹=2, 3⁻¹=1 ✓
Commutative: Yes, so it's Abelian ✓

Group Properties

• Identity is unique
• Each element has unique inverse
• (a⁻¹)⁻¹ = a
• (a * b)⁻¹ = b⁻¹ * a⁻¹ (shoes-socks property)
• Cancellation: a * b = a * c ⟹ b = c

Subgroups

Definition

H ⊆ G is a subgroup of (G, *), written H ≤ G, if (H, *) is itself a group under the same operation.

Subgroup Tests

One-Step Test

H ≤ G iff H ≠ ∅ and
∀a, b ∈ H: a * b⁻¹ ∈ H

Two-Step Test

H ≤ G iff:
1. a, b ∈ H ⟹ a * b ∈ H
2. a ∈ H ⟹ a⁻¹ ∈ H

Finite Subgroup Test

If H is finite, H ≤ G iff
H ≠ ∅ and H is closed under *

Trivial Subgroups

Every group G has:

  • Trivial subgroup: {e} (just identity)
  • Improper subgroup: G itself

Example

Show that 2ℤ = {..., -4, -2, 0, 2, 4, ...} is a subgroup of (ℤ, +).

One-step test:
• 2ℤ ≠ ∅ (contains 0) ✓
• For 2a, 2b ∈ 2ℤ: 2a + (-2b) = 2(a-b) ∈ 2ℤ ✓
Therefore 2ℤ ≤ ℤ

Cyclic Groups

Definition

Group G is cyclic if ∃g ∈ G such that every element can be written as gⁿ for some integer n.
g is called a generator. Write G = ⟨g⟩.

Order of an Element

Definition

Order of a, denoted |a| or ord(a), is the smallest positive integer n such that aⁿ = e.
If no such n exists, order is infinite.

Properties of Cyclic Groups

• Every cyclic group is Abelian
• Subgroups of cyclic groups are cyclic
• If |G| = n, then G ≅ ℤₙ
• If G is infinite cyclic, then G ≅ ℤ
• a is generator of ℤₙ iff gcd(a, n) = 1
• Number of generators of ℤₙ = φ(n) (Euler's totient)

Example: Find Generators

Find all generators of ℤ₈ = {0,1,2,3,4,5,6,7}.

a generates ℤ₈ iff gcd(a, 8) = 1
gcd(1,8)=1 ✓, gcd(3,8)=1 ✓, gcd(5,8)=1 ✓, gcd(7,8)=1 ✓
Generators: {1, 3, 5, 7}
Count = φ(8) = 4 ✓

Example: Order

Find order of 3 in ℤ₁₂.

3¹ = 3, 3² = 6 (mod 12), 3³ = 9, 3⁴ = 12 ≡ 0
ord(3) = 4 (smallest n where 3n ≡ 0 mod 12)
Note: ord(3) = 12/gcd(3,12) = 12/3 = 4 ✓

Cosets & Lagrange's Theorem

Definition: Coset

For H ≤ G and a ∈ G:
Left coset: aH = {ah | h ∈ H}
Right coset: Ha = {ha | h ∈ H}

Coset Properties

• |aH| = |H| (all cosets same size)
• aH = H iff a ∈ H
• aH = bH iff a⁻¹b ∈ H
• Two cosets are either equal or disjoint
• Cosets partition G

Lagrange's Theorem

Theorem

If G is finite and H ≤ G, then |H| divides |G|.

Index: [G : H] = |G|/|H| = number of distinct cosets

Corollaries of Lagrange's Theorem

1. Order of any element divides |G|
2. Groups of prime order are cyclic
3. aᴵᴳᴵ = e for all a ∈ G
4. Prime order groups have no proper subgroups

Example

List cosets of H = {0, 3} in ℤ₆.

0 + H = {0, 3}
1 + H = {1, 4}
2 + H = {2, 5}
3 + H = {3, 0} = H (same as 0+H)

Distinct cosets: {0,3}, {1,4}, {2,5}
[ℤ₆ : H] = 6/2 = 3 ✓

Example: Using Lagrange

Can a group of order 15 have a subgroup of order 6?

By Lagrange, |H| must divide |G|.
Does 6 divide 15? 15/6 = 2.5 (not integer)
No, impossible!

Rings & Fields

Definition: Ring

A ring (R, +, ·) is a set R with two operations where:
R1. (R, +) is an Abelian group
R2. · is associative
R3. Distributive: a·(b+c) = a·b + a·c and (a+b)·c = a·c + b·c

Types of Rings

TypeAdditional PropertyExample
Commutative Ringa · b = b · aℤ, ℤₙ
Ring with UnityHas multiplicative identity 1ℤ, Matrices
Integral DomainCommutative + unity + no zero divisorsℤ
Division RingEvery non-zero has mult. inverseℍ (Quaternions)

Definition: Field

A field (F, +, ·) is a commutative ring where (F - {0}, ·) is also a group.
Equivalently: Every non-zero element has a multiplicative inverse.

Zero Divisors

Definition

In ring R, a ≠ 0 is a zero divisor if ∃b ≠ 0 such that ab = 0.

Example: Zero Divisors in ℤ₆

Find zero divisors in ℤ₆.

2 × 3 = 6 ≡ 0 (mod 6)
So 2 and 3 are zero divisors.
Also: 3 × 4 = 12 ≡ 0 ⟹ 4 is zero divisor
Zero divisors: {2, 3, 4}

Example: ℤ₅ is a Field

Show ℤ₅ = {0, 1, 2, 3, 4} is a field.

5 is prime, so ℤ₅ has no zero divisors.
Multiplicative inverses:
1⁻¹ = 1 (1×1 = 1)
2⁻¹ = 3 (2×3 = 6 ≡ 1)
3⁻¹ = 2 (3×2 = 6 ≡ 1)
4⁻¹ = 4 (4×4 = 16 ≡ 1)
All non-zero have inverses ⟹ Field ✓

Key Result

ℤₙ is a field iff n is prime.
When n is prime, gcd(a, n) = 1 for all 0 < a < n, so all have inverses.
