Propositions

Definition: Proposition

A proposition is a declarative statement that is either TRUE (T/1) or FALSE (F/0), but not both. Also called a statement or assertion.

Examples

✓ Propositions

• "2 + 2 = 4" (True)
• "Paris is in Germany" (False)
• "7 is a prime number" (True)
• "All cats are mammals" (True)

✗ Not Propositions

• "What time is it?" (question)
• "Close the door" (command)
• "x + 1 = 2" (variable, no truth value)
• "This statement is false" (paradox)

Propositional Variables

Notation

We use lowercase letters p, q, r, s... to represent propositions.
Example: Let p = "It is raining", q = "I will carry umbrella"

Understanding Implication (p → q)

Key Insight

p → q is FALSE only when p is TRUE and q is FALSE

Think of it as a promise: "If you get A, I'll buy you a car"
• Get A, get car: Promise kept (T)
• Get A, no car: Promise broken (F)
• No A, get car: Promise not violated (T)
• No A, no car: Promise not violated (T)

Logical Equivalence

Definitions

Tautology: Always TRUE for all truth values (e.g., p ∨ ¬p)
Contradiction: Always FALSE for all truth values (e.g., p ∧ ¬p)
Contingency: Sometimes TRUE, sometimes FALSE
Logical Equivalence: p ≡ q means p ↔ q is a tautology

Important Equivalences (Laws)

Identity: p ∧ T ≡ p, p ∨ F ≡ p
Domination: p ∨ T ≡ T, p ∧ F ≡ F
Idempotent: p ∨ p ≡ p, p ∧ p ≡ p
Double Negation: ¬(¬p) ≡ p
Commutative: p ∨ q ≡ q ∨ p, p ∧ q ≡ q ∧ p
Associative: (p ∨ q) ∨ r ≡ p ∨ (q ∨ r)
Distributive: p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r)
De Morgan: ¬(p ∧ q) ≡ ¬p ∨ ¬q, ¬(p ∨ q) ≡ ¬p ∧ ¬q
Absorption: p ∨ (p ∧ q) ≡ p, p ∧ (p ∨ q) ≡ p
Implication: p → q ≡ ¬p ∨ q
Contrapositive: p → q ≡ ¬q → ¬p
Biconditional: p ↔ q ≡ (p → q) ∧ (q → p)

Example: Simplify

Simplify: ¬(p → q)

¬(p → q)
≡ ¬(¬p ∨ q)   (Implication law)
≡ ¬(¬p) ∧ ¬q   (De Morgan's)
≡ p ∧ ¬q   (Double negation)

Predicate Logic

Definition: Predicate

A predicate P(x) is a statement containing variable x that becomes a proposition when x is assigned a specific value.

Example: P(x) = "x > 5"
P(3) is FALSE, P(7) is TRUE

Quantifiers

Universal Quantifier (∀)

∀x P(x): "For ALL x, P(x) is true"

True if P(x) is true for EVERY x in domain
False if even ONE x makes P(x) false

Existential Quantifier (∃)

∃x P(x): "There EXISTS an x such that P(x)"

True if at least ONE x makes P(x) true
False if NO x makes P(x) true

Negating Quantifiers

¬(∀x P(x)) ≡ ∃x ¬P(x)
"Not all x satisfy P" ≡ "Some x does not satisfy P"

¬(∃x P(x)) ≡ ∀x ¬P(x)
"No x satisfies P" ≡ "All x fail P"

Example

Negate: "All students passed the exam"

Let P(x) = "x passed the exam"
Original: ∀x P(x)
Negation: ¬(∀x P(x)) ≡ ∃x ¬P(x)
"Some student did not pass the exam"

Nested Quantifiers

StatementMeaning
∀x∀y P(x,y)For all x and all y, P(x,y) is true
∀x∃y P(x,y)For every x, there exists some y such that P(x,y)
∃x∀y P(x,y)There exists x such that for all y, P(x,y)
∃x∃y P(x,y)There exist x and y such that P(x,y)

Order Matters!

∀x∃y P(x,y) ≠ ∃y∀x P(x,y)
"Everyone has a mother" ≠ "Someone is everyone's mother"

Rules of Inference

What is Inference?

An argument is a sequence of statements (premises) leading to a conclusion. An argument is valid if the conclusion follows logically from premises.

Rules for Propositional Logic

RuleFormExample
Modus Ponensp→q, p ∴ qIf rain then wet. It rains. ∴ Ground wet.
Modus Tollensp→q, ¬q ∴ ¬pIf rain then wet. Not wet. ∴ No rain.

Rules for Quantified Statements

RuleForm
Universal Instantiation∀x P(x) ∴ P(c) for any c
Universal GeneralizationP(c) for arbitrary c ∴ ∀x P(x)

Example: Valid Argument

Premises: (1) All humans are mortal. (2) Socrates is human.
Conclusion: Socrates is mortal.

Let H(x) = "x is human", M(x) = "x is mortal"
1. ∀x (H(x) → M(x))   (Premise)
2. H(Socrates)   (Premise)
3. H(Socrates) → M(Socrates)   (Universal Instantiation from 1)
4. M(Socrates)   (Modus Ponens from 2, 3)
Valid argument ✓

Proof Methods

Direct Proof

Assume p is true, show q follows.
To prove p → q: Assume p, derive q.

Proof by Contrapositive

Prove ¬q → ¬p instead of p → q.
Assume ¬q, show ¬p follows.

Proof by Contradiction

Assume negation of conclusion.
Derive a contradiction. Original must be true.

Proof by Cases

Split into exhaustive cases.
Prove each case separately.

Example: Direct Proof

Prove: If n is odd, then n² is odd.

Direct Proof:
Assume n is odd.
Then n = 2k + 1 for some integer k.
n² = (2k + 1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1
Let m = 2k² + 2k, then n² = 2m + 1
Therefore n² is odd. ∎

Example: Contradiction

Prove: √2 is irrational.

Proof by Contradiction:
Assume √2 is rational, so √2 = a/b (lowest terms).
Then 2 = a²/b², so a² = 2b².
Thus a² is even, so a is even. Let a = 2k.
(2k)² = 2b² → 4k² = 2b² → b² = 2k²
So b² is even, meaning b is even.
Both a and b even contradicts "lowest terms".
Therefore √2 is irrational. ∎
