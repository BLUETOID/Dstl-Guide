Formula Sheet

Quick Reference for All DSTL Formulas & Key Concepts

Set Theory Formulas

Cardinality Formulas

|A ∪ B| = |A| + |B| - |A ∩ B|
|A ∪ B ∪ C| = |A| + |B| + |C| - |A∩B| - |B∩C| - |A∩C| + |A∩B∩C|
|A - B| = |A| - |A ∩ B|
|A × B| = |A| · |B|
|P(A)| = 2|A|   (Power Set)

Set Laws

Law Union Intersection
Identity A ∪ ∅ = A A ∩ U = A
Domination A ∪ U = U A ∩ ∅ = ∅
Idempotent A ∪ A = A A ∩ A = A
Complement A ∪ A' = U A ∩ A' = ∅
De Morgan's (A ∪ B)' = A' ∩ B' (A ∩ B)' = A' ∪ B'

Relations Formulas

Counting Relations

Total relations from A to B = 2|A|×|B|
Relations on set A = 2|A|²
Reflexive relations on A = 2|A|² - |A|
Symmetric relations on A = 2|A|(|A|+1)/2

Properties of Relations

R
Reflexive: (a,a) ∈ R ∀a ∈ A
S
Symmetric: (a,b) ∈ R ⟹ (b,a) ∈ R
T
Transitive: (a,b),(b,c) ∈ R ⟹ (a,c) ∈ R
A
Anti-symmetric: (a,b),(b,a) ∈ R ⟹ a=b
Equivalence Relation = Reflexive + Symmetric + Transitive
Partial Order = Reflexive + Anti-symmetric + Transitive

Closures

Reflexive Closure: r(R) = R ∪ IA
Symmetric Closure: s(R) = R ∪ R-1
Transitive Closure: t(R) = R ∪ R² ∪ R³ ∪ ... ∪ Rn

Functions

Counting Functions

Total functions from A to B = |B||A|
Injective (one-to-one) = P(|B|, |A|) = |B|!/(|B|-|A|)!
Bijective = |A|!  (when |A| = |B|)

Composition

(g ∘ f)(x) = g(f(x))
Apply f first, then apply g to the result

Propositional Logic

Logical Connectives Truth Table

p q ¬p p ∧ q p ∨ q p → q p ↔ q p ⊕ q
TTFTTTTF
TFFFTFFT
FTTFTTFT
FFTFFTTF

Important Equivalences

p → q ≡ ¬p ∨ q
p ↔ q ≡ (p → q) ∧ (q → p)
¬(p → q) ≡ p ∧ ¬q
Contrapositive: p → q ≡ ¬q → ¬p
De Morgan's: ¬(p ∧ q) ≡ ¬p ∨ ¬q
De Morgan's: ¬(p ∨ q) ≡ ¬p ∧ ¬q

Rules of Inference

MP
Modus Ponens:
p, p→q ⊢ q
MT
Modus Tollens:
¬q, p→q ⊢ ¬p
HS
Hypothetical Syllogism:
p→q, q→r ⊢ p→r
DS
Disjunctive Syllogism:
p∨q, ¬p ⊢ q
∧I
Conjunction:
p, q ⊢ p ∧ q
∨I
Addition:
p ⊢ p ∨ q

Predicate Logic

Quantifier Negation

¬∀x P(x) ≡ ∃x ¬P(x)
¬∃x P(x) ≡ ∀x ¬P(x)

Quantifier Distribution

∀x (P(x) ∧ Q(x)) ≡ ∀x P(x) ∧ ∀x Q(x)
∃x (P(x) ∨ Q(x)) ≡ ∃x P(x) ∨ ∃x Q(x)
Warning: ∀ does NOT distribute over ∨, and ∃ does NOT distribute over ∧

Boolean Algebra

Boolean Laws

Law + (OR) · (AND)
Identity A + 0 = A A · 1 = A
Null A + 1 = 1 A · 0 = 0
Idempotent A + A = A A · A = A
Complement A + A' = 1 A · A' = 0
Absorption A + AB = A A(A + B) = A
De Morgan's (A + B)' = A'B' (AB)' = A' + B'

K-Map Grouping Rules

1
Groups must be powers of 2 (1, 2, 4, 8...)
2
Larger groups = simpler expression
3
Groups can wrap around edges
4
Every 1 must be in at least one group
SOP: Group 1s, POS: Group 0s (then complement variables)

Lattices & Posets

Lattice Definitions

∨
Join (LUB): Least Upper Bound of a,b
∧
Meet (GLB): Greatest Lower Bound of a,b
Lattice: A poset where every pair has both LUB and GLB

Divisibility Lattice Dn

GLB(a, b) = GCD(a, b)
LUB(a, b) = LCM(a, b)
|Dn| = Number of divisors of n = τ(n)

Group Theory

Group Axioms

C
Closure: a * b ∈ G
A
Associativity: (a*b)*c = a*(b*c)
I
Identity: ∃e: a*e = e*a = a
N
Inverse: ∃a⁻¹: a*a⁻¹ = e

Key Theorems

Lagrange's Theorem: |H| divides |G|
Index: [G : H] = |G| / |H|
Euler's Theorem: aφ(n) ≡ 1 (mod n), gcd(a,n)=1
Fermat's Little: ap-1 ≡ 1 (mod p), p prime

Cyclic Groups

Number of generators of ℤn = φ(n)
Order of ak in ℤn = n / gcd(k, n)
a is a generator iff gcd(a, n) = 1

Euler's Totient Function

φ(p) = p - 1  (p prime)
φ(pk) = pk - pk-1 = pk-1(p - 1)
φ(mn) = φ(m) · φ(n)  (gcd(m,n) = 1)

Graph Theory

Basic Formulas

Handshaking Lemma: Σ deg(v) = 2|E|
Number of odd-degree vertices is always EVEN

Complete Graphs Kn

Vertices = n
Edges = n(n-1)/2 = C(n,2)
Degree of each vertex = n - 1

Trees

Tree with n vertices: Edges = n - 1
Cayley's Formula: Labeled trees on n vertices = nn-2

Planar Graphs - Euler's Formula

V - E + F = 2
For connected planar graph: E ≤ 3V - 6
If no triangles: E ≤ 2V - 4
Kuratowski: Non-planar iff contains K₅ or K₃,₃ subdivision

Graph Coloring

χ(Kn) = n
χ(Cn) = 2 if n even, 3 if n odd
Four Color Theorem: χ(planar) ≤ 4

Euler & Hamiltonian Paths

Type Condition
Eulerian Circuit All vertices have even degree
Eulerian Path Exactly 2 vertices have odd degree
Hamiltonian No simple necessary/sufficient condition
Dirac's Theorem: If deg(v) ≥ n/2 for all v, then Hamiltonian

Combinatorics Quick Reference

Counting Formulas

Permutation: P(n,r) = n! / (n-r)!
Combination: C(n,r) = n! / [r!(n-r)!]
With Repetition: n! / (n₁! · n₂! · ... · nₖ!)
Pigeonhole: ⌈n/k⌉ items in some box
